# GitHub 公开库发布前检查

## 仓库内容

- 不包含私有客户资料、报价单、合同、账号、密码或服务器配置。
- 不包含未确认可以公开的内部数据源。
- 不包含压缩包、临时文件或历史污染文件。
- 不包含恶意代码、漏洞利用样例或隐藏脚本。

## 页面行为

- 无自动跳转。
- 无隐藏 iframe。
- 无收集表单。
- 无第三方跟踪脚本。
- 无短链或伪装链接。

## 导流方式

- 官网链接有上下文说明。
- 页面不是纯外链列表。
- 页面提供选型说明、应用解释和精选型号。
- 联系方式与主站一致。

## GitHub Pages

- `index.html` 位于仓库根目录。
- 所有资源使用相对路径。
- 不依赖构建工具。
- 不依赖外部 CDN。

## 编码

- 所有文件为 UTF-8 without BOM。
- 无 replacement character。
- 无已知 mojibake。
